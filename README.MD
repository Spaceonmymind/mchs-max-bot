## Бот предоставляет:

* инструкции при ЧС
* справку о службах
* ответы на частые вопросы
* удобную систему категорий и тем

Проект написан на Node.js, упакован в Docker и поддерживает развёртывание через `docker-compose`.

## Возможности бота

* автоматическое приветствие при первом запуске (`bot_started`)
* загрузка логотипа и отправка в виде вложения
* структурированное меню из разделов
* быстрый переход по темам
* кнопка «Назад» внутри навигации
* чистая архитектура по директориям (`handlers`, `keyboards`, `utils`)

## Структура проекта

## Запуск через Docker

### 1. Установить Docker и Docker Compose

Для Linux:

```
sudo apt install docker.io docker-compose
```

### 2. Клонировать проект

```
git clone <репозиторий>
cd mchs-max-bot
```

### 3. Создать `.env`

```
nano .env
```

Вставить токен.

### 4. Запустить бота

```
docker-compose up -d --build
```

Контейнер поднимется с auto-restart.

---

## Обновление бота

При выходе новой версии:

```
git pull
docker-compose up -d --build
```

Контейнер пересоберётся и перезапустится.

## Логирование

Логи контейнера:

```
docker logs -f mchs_max_bot
```


## Основная логика

### `bot_started`

Отправляет логотип, приветственный текст и вызывает стартовое меню.

### `startHandler`

Выводит категории вопросов.

### `categoryHandler`

Обрабатывает выбор раздела.

### `topicHandler`

Показывает конкретный ответ.
